# Checkout Friction Reduction

1. Baseline checkout funnel metrics by step, device, and traffic source.
2. Identify highest-friction form fields and interaction bottlenecks.
3. Simplify required inputs and reduce cognitive load.
4. Improve trust elements (security cues, guarantees, transparent totals).
5. Optimize payment options, error handling, and mobile usability.
6. Define recovery triggers for partial checkouts.
7. Output prioritized fixes with expected conversion lift.
