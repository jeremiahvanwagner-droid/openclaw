# Fundamental Data Extraction

1. Detect new filings/releases from approved sources.
2. Parse documents into structured financial fields.
3. Validate extracted values against schema and consistency checks.
4. Reconcile revisions and restatements by source version.
5. Compute derived features for model consumption.
6. Publish updates with provenance metadata.
7. Output extraction quality and freshness report.
