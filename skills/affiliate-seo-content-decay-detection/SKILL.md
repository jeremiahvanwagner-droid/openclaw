# Content Decay Detection

1. Monitor URL-level performance trends (rank, traffic, CTR, conversion).
2. Detect sustained decline against historical baseline.
3. Attribute likely decay causes (outdated facts, weak intent match, stale SERP features).
4. Select refresh scope by impact and effort.
5. Draft replacement paragraphs for decayed sections.
6. Queue updates with single-page change tracking.
7. Output refresh backlog with expected lift estimates.
