# Portfolio Rebalancing

1. Compare current holdings versus target allocation model.
2. Identify drift, concentration breaches, and risk imbalances.
3. Generate trade set minimizing cost and tax impact.
4. Enforce turnover, liquidity, and exposure constraints.
5. Sequence orders to reduce market impact.
6. Verify post-trade target proximity and residual risk.
7. Output rebalance report with attribution.
