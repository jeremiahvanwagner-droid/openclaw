# Custom Capability Mapping

1. Define competency framework and proficiency levels.
2. Gather evidence from tasks, outcomes, and behaviors.
3. Score current capability state by domain.
4. Identify capability gaps blocking next-stage performance.
5. Recommend development actions for each gap.
6. Track capability progression over time.
7. Output capability matrix and growth roadmap.
