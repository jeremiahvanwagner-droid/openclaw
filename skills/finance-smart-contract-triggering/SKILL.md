# Smart Contract Triggering

1. Validate contract allowlist, chain, and function permissions.
2. Simulate transaction outcome before signing.
3. Check gas, slippage, and MEV-risk thresholds.
4. Enforce signer policy and multi-step approval requirements.
5. Submit transaction with nonce and replay protections.
6. Track confirmation state and reorg-safe finality rules.
7. Output transaction audit log with risk annotations.
