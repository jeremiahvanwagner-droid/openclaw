# Interactive Assignment Grading

1. Load assignment rubric, competency targets, and submission format rules.
2. Evaluate submission against objective and qualitative criteria.
3. Produce criterion-level scores with evidence-backed comments.
4. Detect edge cases requiring human review escalation.
5. Deliver instant formative feedback with revision guidance.
6. Store grades and rationale for auditability.
7. Output learner-facing summary and instructor analytics.
