# Lead Magnet Generation

1. Define the target audience, quick-win outcome, and funnel entry intent.
2. Select the best lead magnet format based on implementation time and perceived value.
3. Build a concise structure with title, promise, steps, and action checklist.
4. Format for delivery channel requirements (PDF, email body, landing block, download page).
5. Add clear CTA paths to the next ladder step.
6. Validate tone, compliance, and claim safety.
7. Output publish-ready assets plus a shorter alternate variant.
