# Micro-lesson Atomization

1. Segment source lecture into concept boundaries and learning objectives.
2. Extract one clear outcome per micro-lesson.
3. Rewrite content into short, focused instructional units.
4. Attach quick checks and action prompts to each unit.
5. Sequence micro-lessons by prerequisite logic.
6. Tag assets for searchability and competency linkage.
7. Output publish-ready micro-lesson bundle with metadata.
