# Codebase Dependency Resolution

1. Scan dependency graph for vulnerabilities and deprecations.
2. Classify updates by risk level and compatibility impact.
3. Propose minimal-safe upgrade paths.
4. Apply updates in controlled batches.
5. Run tests and static checks after each batch.
6. Roll back failing updates and document blockers.
7. Output upgrade report with residual risk list.
