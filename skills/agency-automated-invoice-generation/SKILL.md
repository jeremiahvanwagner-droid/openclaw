# Automated Invoice Generation

1. Pull billing terms, milestones, and approved billable events.
2. Calculate invoice line items with tax and discount rules.
3. Validate invoice against contract constraints.
4. Generate invoice artifact and payment instructions.
5. Dispatch invoice via configured channel.
6. Track payment status and trigger reminders on schedule.
7. Output receivables report and overdue escalation queue.
