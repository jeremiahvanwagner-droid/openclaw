# Autonomous Debugging

1. Ingest error logs, traces, and failing test evidence.
2. Cluster incidents by root-cause signature.
3. Generate candidate fixes with minimal blast radius.
4. Validate patches in isolated test environments.
5. Apply approved fixes with rollback safeguards.
6. Monitor post-fix stability and regression signals.
7. Output incident resolution report and prevention notes.
