# Hierarchical Consensus Building

1. Collect independent recommendations from approved model layers.
2. Apply deterministic rule-engine validation to each proposal.
3. Reconcile disagreements using tie-break and confidence rules.
4. Require escalation for unresolved high-impact conflicts.
5. Produce consensus decision with rationale trace.
6. Block execution if consensus threshold is not met.
7. Output decision packet for audit and replay.
