# Schema Markup Generation

1. Detect page type and applicable schema classes.
2. Build JSON-LD blocks from page content and metadata.
3. Validate required/recommended properties per schema type.
4. Ensure consistency between visible content and markup claims.
5. Inject schema into publish pipeline safely.
6. Run validation checks and capture errors.
7. Output deployment log with coverage metrics.
