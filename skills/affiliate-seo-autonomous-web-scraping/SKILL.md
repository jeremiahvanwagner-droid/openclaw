# Autonomous Web Scraping

1. Define approved target domains and data fields.
2. Configure crawl strategy for static and dynamic pages.
3. Extract pricing, offer attributes, and affiliate-related metadata.
4. Normalize outputs into canonical schema.
5. Deduplicate and validate data quality.
6. Respect robots/policy constraints and rate limits.
7. Output structured dataset with timestamped snapshots.
