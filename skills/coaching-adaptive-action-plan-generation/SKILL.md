# Adaptive Action Plan Generation

1. Capture current objective, blocker type, and constraint context.
2. Diagnose root cause of stalled execution.
3. Generate alternative action paths with reduced friction.
4. Prioritize options by impact, feasibility, and urgency.
5. Convert selected path into concrete next steps.
6. Add checkpoint triggers for early feedback.
7. Output revised plan with rationale and timeline.
