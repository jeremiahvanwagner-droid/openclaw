# Omnichannel Routing

1. Resolve member channel preferences and delivery permissions.
2. Classify announcement priority and urgency tier.
3. Format channel-specific message variants from one source payload.
4. Dispatch with deduplication and fallback channel logic.
5. Track delivery, open, and interaction confirmations.
6. Retry failed sends within bounded retry policies.
7. Output delivery coverage report and unresolved recipients.
