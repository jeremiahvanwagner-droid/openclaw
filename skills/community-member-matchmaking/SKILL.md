# Member Matchmaking

1. Collect member goals, interests, skill levels, and availability.
2. Build compatibility scores using weighted matching criteria.
3. Filter out low-fit or policy-restricted pairings.
4. Generate recommended 1:1 or small-group matches.
5. Create introduction prompts and kickoff guidelines.
6. Track match engagement and satisfaction signals.
7. Recycle unmatched members into next matching cycle.
