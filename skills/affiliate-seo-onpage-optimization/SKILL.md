# On-page SEO Optimization

1. Audit title, headings, metadata, and content-intent alignment.
2. Improve semantic coverage and entity depth.
3. Optimize internal links, anchors, and topical context.
4. Validate media, alt text, and page experience signals.
5. Remove keyword stuffing and over-optimization artifacts.
6. Align calls-to-action with user intent stage.
7. Output optimization checklist and before/after diff summary.
