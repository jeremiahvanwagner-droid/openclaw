# Dynamic Description Generation

1. Collect SKU specs, use cases, audience signals, and brand voice constraints.
2. Build keyword intent set for search relevance.
3. Draft benefit-led product copy with feature-to-outcome mapping.
4. Generate metadata elements (title, meta description, bullets, alt text).
5. Validate claims, compliance, and readability.
6. Produce variant copy by channel or locale if required.
7. Output publish-ready description package with SEO notes.
