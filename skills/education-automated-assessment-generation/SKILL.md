# Automated Assessment Generation

1. Parse current learning objectives, competencies, and lesson scope.
2. Select assessment formats by objective type (recall, application, synthesis).
3. Generate item pools with varied difficulty and distractor quality.
4. Validate item clarity, answer correctness, and bias constraints.
5. Tag questions by competency and cognitive level.
6. Assemble assessments with balanced coverage and timing.
7. Output graded answer key and remediation mapping.
