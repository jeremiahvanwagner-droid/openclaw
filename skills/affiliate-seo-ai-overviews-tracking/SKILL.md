# AI Overviews Tracking

1. Monitor priority queries for AI overview presence and citation patterns.
2. Capture formatting, source types, and response structures being cited.
3. Evaluate owned content alignment against observed inclusion patterns.
4. Recommend structural edits (concise answers, definitions, list formatting, evidence blocks).
5. Prioritize updates by query value and inclusion probability.
6. Track post-update inclusion outcomes.
7. Output AI overview readiness scorecard.
