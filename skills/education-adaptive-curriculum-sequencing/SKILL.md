# Adaptive Curriculum Sequencing

1. Read learner profile, prior performance, and module dependency graph.
2. Detect unmet prerequisites and mastery confidence by competency.
3. Re-rank upcoming modules by urgency, dependency order, and expected impact.
4. Insert remediation modules before advanced content when gaps are detected.
5. Preserve pacing constraints and required compliance modules.
6. Publish updated sequence with rationale per reorder decision.
7. Track post-adjustment outcomes and refine sequencing policy.
