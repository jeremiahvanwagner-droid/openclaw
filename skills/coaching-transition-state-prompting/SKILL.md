# Transition State Prompting

1. Detect transition points associated with drop-off risk.
2. Classify transition type (cognitive, emotional, logistical).
3. Generate concise micro-prompts for the immediate next step.
4. Deliver prompts at context-appropriate timing.
5. Reinforce progress with low-friction follow-ups.
6. Track transition completion outcomes.
7. Output transition effectiveness insights.
