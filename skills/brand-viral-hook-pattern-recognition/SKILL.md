# Viral Hook Pattern Recognition

1. Collect historical post-level performance and opening lines.
2. Extract recurring hook structures from top performers.
3. Cluster hooks by audience segment and intent type.
4. Generate new hook variants following validated patterns.
5. Filter for brand fit and claim safety.
6. Pair hooks with format-specific content templates.
7. Output hook library with test-ready ideas.
