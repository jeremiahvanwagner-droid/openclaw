# Client Readiness Analysis

1. Define readiness criteria for next-stage strategy.
2. Evaluate foundational mastery evidence.
3. Score readiness confidence and risk level.
4. Identify missing prerequisites and weak spots.
5. Recommend hold/advance decision with justification.
6. Generate prerequisite completion plan where needed.
7. Output readiness assessment report.
