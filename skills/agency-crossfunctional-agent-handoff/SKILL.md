# Cross-functional Agent Handoff

1. Capture task state, history, decisions, and unresolved questions.
2. Package handoff context into standardized transfer payload.
3. Validate required artifacts before transfer.
4. Route handoff to best-fit receiving agent/team.
5. Confirm receipt and understanding acknowledgment.
6. Track handoff completion and reopen loop if context gaps appear.
7. Output handoff audit trail and latency metrics.
