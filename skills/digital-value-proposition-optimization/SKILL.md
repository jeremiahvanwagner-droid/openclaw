# Value Proposition Optimization

1. Capture the current offer statement, ICP, and promised outcome.
2. Extract top customer pains, desired gains, and purchase objections.
3. Compare competing alternatives and identify undifferentiated claims.
4. Generate positioning variants that strengthen clarity, specificity, and proof alignment.
5. Score each variant for relevance, uniqueness, credibility, and conversion potential.
6. Select test candidates and map each to a single-variable messaging experiment.
7. Output recommended primary proposition, backup variant, and deployment locations.
