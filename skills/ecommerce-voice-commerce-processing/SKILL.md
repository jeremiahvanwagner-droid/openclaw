# Voice Commerce Processing

1. Transcribe voice input and extract commerce intent entities.
2. Resolve product references, quantity, and option constraints.
3. Confirm ambiguous requests with concise clarification prompts.
4. Execute actions (search, add-to-cart, modify-cart, checkout) safely.
5. Enforce authentication and payment confirmation policies.
6. Return spoken confirmations with next-step options.
7. Output interaction logs for quality and intent-model tuning.
