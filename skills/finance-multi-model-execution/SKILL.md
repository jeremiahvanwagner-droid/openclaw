# Multi-model Execution

1. Route decision requests to diverse approved model providers.
2. Normalize outputs into shared decision schema.
3. Compare agreement, confidence, and policy alignment.
4. Trigger adjudication rules for divergent outcomes.
5. Select execution path only after thresholded verification.
6. Record model contribution and fallback events.
7. Output robustness score and execution trace.
