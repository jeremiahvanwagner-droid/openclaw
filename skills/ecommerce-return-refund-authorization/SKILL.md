# Return/Refund Authorization

1. Validate order eligibility against return window and product policy.
2. Classify return reason and fraud-risk indicators.
3. Approve, partially approve, or deny based on rules.
4. Generate return shipping label and customer instructions.
5. Trigger refund workflow by payment method and inspection status.
6. Update order, inventory, and finance records.
7. Output return analytics and policy exception report.
