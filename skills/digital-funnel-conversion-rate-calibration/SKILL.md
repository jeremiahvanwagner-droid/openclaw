# Funnel Conversion Rate Calibration

1. Baseline current conversion metrics by stage and device.
2. Identify the highest-friction page element causing drop-off.
3. Propose one-variable A/B hypotheses (headline, CTA, social proof, form length, layout).
4. Define test setup, sample requirements, and success threshold.
5. Run tests in sequence, not in overlapping confounded batches.
6. Evaluate statistical confidence and practical lift.
7. Promote winner, document learnings, and queue next test.
