# Real-time Skill Gap Detection

1. Ingest recent assessment data, assignment outcomes, and interaction logs.
2. Map each error or hesitation signal to a defined competency.
3. Score gap severity using frequency, recency, and impact on progression.
4. Separate transient confusion from persistent deficiency.
5. Trigger targeted remediation recommendations per high-severity gap.
6. Notify orchestration systems with machine-readable gap summaries.
7. Re-evaluate after intervention and update learner gap status.
