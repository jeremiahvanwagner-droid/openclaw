# Review Sentiment Extraction

1. Ingest review text, ratings, and product metadata.
2. Classify sentiment and emotional intensity by aspect.
3. Extract recurring issue themes and defect entities.
4. Quantify defect frequency and severity trends.
5. Map findings to SKU/vendor cohorts.
6. Alert sourcing team for threshold-breaching defect patterns.
7. Output weekly quality insight brief with action priorities.
