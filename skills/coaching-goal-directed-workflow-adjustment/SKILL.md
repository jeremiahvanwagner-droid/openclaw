# Goal-directed Workflow Adjustment

1. Compare current workflow against updated client priorities.
2. Identify misaligned tasks and obsolete workstreams.
3. Re-sequence tasks to protect highest-value outcomes.
4. Reallocate effort across near-term and long-term goals.
5. Update milestones and review cadence.
6. Communicate changes with clear reasoning.
7. Output adjusted workflow map and checkpoint plan.
