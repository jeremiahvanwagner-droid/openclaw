# Session Transcript Extraction

1. Ingest transcript and session agenda context.
2. Extract key decisions, actions, owners, and deadlines.
3. Identify unresolved questions and risk items.
4. Format deliverables into client-ready summary.
5. Sync action items to tracking systems.
6. Flag ambiguities requiring coach confirmation.
7. Output final session recap and execution checklist.
