# Automated Event Scheduling

1. Define event objectives, audience segment, and cadence rules.
2. Collect timezone and availability preferences from member data.
3. Select optimal times based on attendance probability.
4. Reserve slots, publish event metadata, and issue invitations.
5. Send timed reminders and pre-event preparation prompts.
6. Handle updates, cancellations, and waitlist promotion.
7. Output attendance report and follow-up task queue.
