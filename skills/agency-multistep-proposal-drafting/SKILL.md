# Multi-step Proposal Drafting

1. Ingest call notes, client goals, constraints, and success criteria.
2. Map needs to service modules and delivery scope.
3. Build phased proposal structure with outcomes and timelines.
4. Generate pricing options and value justification.
5. Add assumptions, dependencies, and approval checkpoints.
6. Produce concise executive summary and CTA.
7. Output send-ready proposal with shorter alternative variant.
