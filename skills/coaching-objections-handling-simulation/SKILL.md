# Objections Handling Simulation

1. Select objection scenarios by buyer type and deal stage.
2. Run interactive role-play with escalating difficulty.
3. Evaluate response quality across clarity, empathy, and persuasion.
4. Provide immediate corrective coaching after each round.
5. Introduce curveball objections for resilience training.
6. Score progression across repeated simulations.
7. Output performance summary and focused practice plan.
