# Content Gap Analysis

1. Inventory current product modules, promises, and outcomes.
2. Compare expected customer journey against existing content coverage.
3. Identify high-impact missing components (clarity gaps, implementation gaps, proof gaps).
4. Prioritize gaps by revenue impact, support burden, and completion risk.
5. Recommend update candidates with effort and timeline estimates.
6. Map each gap to measurable success criteria.
7. Output ranked backlog for next release cycle.
