# Direct Message Qualification

1. Classify incoming DMs by intent category.
2. Filter spam, low-value pitches, and irrelevant outreach.
3. Score high-intent opportunities by strategic value.
4. Route qualified inquiries to priority review queue.
5. Generate suggested response drafts by inquiry type.
6. Track conversion outcomes from qualified leads.
7. Output inbox triage summary and escalation list.
