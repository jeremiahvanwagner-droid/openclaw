# Asynchronous Progress Tracking

1. Define milestone map, success criteria, and evidence requirements.
2. Ingest client updates from approved channels.
3. Score milestone completion and momentum trends.
4. Detect stalls, regressions, and missed commitments.
5. Trigger update reminders where reporting gaps appear.
6. Summarize progress deltas since last review.
7. Output client status board with next actions.
