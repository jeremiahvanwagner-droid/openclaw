# Dynamic Link Injection

1. Parse content and identify monetizable intent segments.
2. Match candidate anchors to relevant affiliate offers.
3. Apply placement rules for density, readability, and compliance.
4. Insert links with tracking parameters and attribution tags.
5. Validate no broken links or over-optimization signals.
6. A/B test alternative anchors and placements.
7. Output link performance report and optimization queue.
