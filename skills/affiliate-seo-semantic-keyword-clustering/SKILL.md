# Semantic Keyword Clustering

1. Ingest keyword corpus with volume, intent, and difficulty metrics.
2. Generate semantic embeddings and similarity graph.
3. Form intent-aligned keyword clusters.
4. Map clusters to pillar/supporting page hierarchy.
5. Detect cannibalization and overlap risks.
6. Recommend URL structure and internal link paths.
7. Output architecture blueprint with content priorities.
