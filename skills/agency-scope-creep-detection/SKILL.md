# Scope Creep Detection

1. Parse incoming requests and map to contracted scope.
2. Classify request as in-scope, borderline, or out-of-scope.
3. Estimate effort and timeline impact of non-scoped work.
4. Generate recommended change-order language and pricing.
5. Alert account lead for review and client response.
6. Track accepted/rejected scope changes.
7. Output scope variance report with margin impact.
