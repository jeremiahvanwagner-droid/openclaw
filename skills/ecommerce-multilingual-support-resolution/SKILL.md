# Multi-lingual Support Resolution

1. Detect incoming language and classify support intent.
2. Translate request context while preserving entity accuracy.
3. Map issue type to approved policy workflows.
4. Generate response in customer language with localized tone.
5. Handle escalations for sensitive or ambiguous cases.
6. Sync ticket actions across support and order systems.
7. Output resolution metrics by language and issue type.
