# Headless Browser Automation

1. Launch controlled browser sessions with target-specific settings.
2. Navigate dynamic flows and wait for required render states.
3. Execute interactions needed to reveal data.
4. Extract structured fields from rendered DOM.
5. Handle pagination, lazy loading, and anti-bot-safe pacing.
6. Capture fallback screenshots/logs for failed runs.
7. Output validated extraction artifacts and run diagnostics.
