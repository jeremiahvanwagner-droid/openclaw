# Autonomous Client Onboarding

1. Trigger onboarding workflow from signed deal event.
2. Collect required client intake data and missing prerequisites.
3. Provision project spaces, communication channels, and tool access.
4. Apply service package template and kickoff checklist.
5. Schedule onboarding milestones and stakeholder introductions.
6. Validate readiness gates before delivery start.
7. Output onboarding status report with blockers and next actions.
